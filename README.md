# CELPIP Study Guide

Personal comprehensive study guide for the Canadian English Language Proficiency Index Program (CELPIP) exam.

## Sections

- **Home** — Overview, test structure, score quick reference
- **Listening** — 6 parts, question types, strategies, sample question
- **Reading** — 4 parts, question types, time strategy, sample
- **Writing** — Rubric, Task 1 (email), Task 2 (survey), phrases bank
- **Speaking** — 8 tasks with structure guides and useful phrases
- **Scoring** — Full CLB/score chart, CRS points table, descriptors
- **Immigration** — Express Entry, citizenship, PNP, professional licensing

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide icons
